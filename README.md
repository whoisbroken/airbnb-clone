# Airbnb Clone

I'ts a project from freeCodeCamp.org React.js course.

## Installation
Clone this repository, and run:
```bash
npm install
```

Use the command :

```bash
npm start
```
Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.

## Dependencies
* React.js
* gh-pages (for deployment)
